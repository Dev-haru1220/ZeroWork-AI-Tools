# ZeroWork-Tools 🚀
> **Windowsの上に乗る、実務特化型の「自動化OS」プロトタイプ**

<img src="https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3742490/deb4c09e-5ba8-4e13-a646-0c550bdda2dc.png" width="300" alt="ZeroWork-Tools Icon">

ZeroWork-Toolsは、単なるAIアプリではありません。最新のVision LLM（Gemini 2.5 Flash）を核に、Windowsデスクトップ上のあらゆるアナログ作業をデジタル資産へと変錬する、次世代の**「自動化パイプライン」**です。

「AIに仕事を奪われる」のではなく、**「AIを最新鋭のツールとして使いこなし、紙やPDFの鎖から解放される」**ためのインフラを提供します。

---

## 💎 5つの核心機能（The Five Artifacts）

現在はMVPとして以下の5機能を軸に展開し、今後数百におよぶ実務自動化モジュールを統合予定です。

| 機能 | 役割 | 概要 |
| :--- | :--- | :--- |
| **① AIインテリジェント・整理** | **「決断」の自動化** | デスクトップや共有フォルダをAIが常時監視。中身を理解し、社内ルールに基づき自動リネーム・仕分けを完結させます。 |
| **② 万能データ・トランスミッター** | **「転記」の自動化** | 画像・PDF・スクショから「意味」を抽出。インボイス(T番号)等を含む構造化データをExcelへ直接流し込みます。 |
| **③ Excelリモート・マジック** | **「思考」の自動化** | 言語指示により、起動中のExcelを直接操作。グラフ作成、関数入力、書式設定をAIがユーザーの目の前で実行します。 |
| **④ 右クリック・ユーティリティ** | **「作業」の自動化** | Windows標準メニューを拡張。ブラウザを介さず、ファイル変換や圧縮解凍を右クリック1秒で完了させます。 |
| **⑤ スマート・ワークログ** | **「教育」の自動化** | ユーザーやチームの整理ルールを学習。新人が入っても「AIがルールを教える」自律型の組織知を構築します。 |

---

## 🛠 テクニカル・スタック

- **Core Engine**: Google Gemini 2.5 Flash (Vision LLM)
- **Language**: Python 3.14+
- **Infrastructure**: Google GenAI SDK, openpyxl, watchdog
- **Target OS**: Windows 11 (Desktop Native)
